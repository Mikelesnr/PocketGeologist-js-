About Pocket Geologist

Pocket Geologist is a personal project developed as part of the WDD330 - Frontend Development II course. It is designed to help users explore and manage their own collection of minerals while utilizing data from the Mindat.org API.

Project Goals

  • Create an intuitive frontend for mineral collection management.
  • Fetch mineral data using the Mindat.org API.
  • Implement user authentication for personalized collections.
  • Ensure a responsive and modular UI design.

Technologies Used

  • JavaScript (ES6+): Core language for application logic.
  • Vite: Fast frontend development framework.
  • LocalStorage: For user data persistence.
  • Mindat API: Source for mineral data.
  • CSS (Responsive): Enhancing styling and layout.
